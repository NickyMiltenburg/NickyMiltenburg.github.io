Binair stelsel als code

In Nederland kennen we het tientallig getallenstelsel, met de cijfers 0 tot en met 9. Naast deze cijfers hebben we ook het binair stelsel.
In het binair stelsel werken ze met een tweetallig getallenstelsel, deze bestaan uit de cijfers 0 en 1, ook wel een bit genoemd.
Door deze 2 cijfers in een bepaalde volgorde te zetten gaan deze cijfers getallen betekenen, deze getallen zijn een code voor de computer.
Er worden dus instructies aan de computer gegeven het behulp van de 2 cijfers (0 en 1)

Hoe berekenen we het binair stelsel?

Geheugencellen van een computer kunnen maar twee waarden aannemen, waardoor we het binair stelsel gebruiken.
Wanneer je een binair getal wil omrekenen naar een decimaal getal, is het belangrijk om te kijken naar de plek waar een 1 staat.
Voor ieder binair cijfer berekenen we de waarde door te kijken wat de plek van dit cijfer is, tellende van rechts. Vanaf daar trek je er 1 van af en neem je de macht van twee.
Als je een getal wil omzetten naar een binair getal deel je het getal door 2 en schrijf je de rest op. Je moet de resten van rechts naar links opschrijven en dan heb je het binaire getal.

Een voorbeeld

Een voorbeeld van een binair getal is bijvoorbeeld: 10110.
Elk cijfer in dit binaire getal heeft een andere positie waarde. We tellen vanuit rechts:
Het laatste cijfer is een 0, dit staat voor 2^0 = 1
Het een na laatste plekje, dus een plekje naar links geschoven is een 1, dus 2^1 = 2
Het volgende cijfer is een 1, dus 2^2 = 4
Het cijfer daarna is een 0, dus 2^3 = 8
Het laatste cijfer is een 1, dus 2^4 = 16
